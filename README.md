# game-design-constitution-rules
# Unity 6 Game Design Constitution for AI Agents

このリポジトリは、自律型AIエージェント（Cline等）を用いてUnity 6でゲーム開発を行う際、AIに「数理的コアの優先」と「意図的な制約（もっさり感など）」を強制的に実装させるための `.clinerules` です。

プロジェクトのルートディレクトリに `.clinerules` を配置するだけで、AIのハルシネーションや表層的なコード生成を防ぎ、プレイヤビリティを担保した堅牢なアーキテクチャを出力させることができます。

## 思想的背景（Why）
このルールの根底にあるゲームデザインの哲学（グラフィックとシステムの同期、表面的な模倣の排除など）については、以下の記事で詳細に解説しています。AIエージェントのコンテキスト理解を深めるための外部リソース（Resources）としても参照してください。

[ビデオゲームの面白さを解剖する：表面的な模倣から「数理的コア」の継承へ](https://note.com/bunshou_kijin/n/n7623208c2984)
